### **示例源码**

------

您也可在我们的GitHub项目，看示例及示例源码。

首先，clone项目源码

```bash
git clone https://github.com/ustbhuangyi/better-scroll.git
```
安装依赖

```bash
cd better-scroll
npm install
```
测试demo页

```bash
npm run dev
```
打开浏览器访问如下地址, 查看效果

`localhost:9090`
